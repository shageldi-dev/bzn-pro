
## Description


## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run migrations

```bash
# for running all migrations
$ npm run db:seed

# for running single migration by classname
$ npm run db:seedone `MigrationClassName`
```
